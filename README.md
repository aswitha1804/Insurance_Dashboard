# 🛡️ Insurance SafeGuard - Analytics Dashboard

A professional, enterprise-grade insurance analytics portal with real-time claims data visualization, agent performance tracking, and IBM Cognos integration.

## ✨ Features

- 🔐 **Secure Login** - Basic authentication with validation
- 📊 **Interactive Dashboard** - Real-time KPI metrics with Chart.js
- 🌓 **Dark/Light Mode** - Toggle themes with persistent storage
- 📱 **Fully Responsive** - Works on all devices (Desktop, Tablet, Mobile)
- 🎨 **Modern UI** - Glassmorphism effects, gradients, and smooth animations
- 🚀 **IBM Cognos Integration** - Launch full analytics dashboard in new tab
- 📈 **Data Visualizations** - Bar charts, line charts, doughnut charts, and radar charts
- 📞 **Contact Page** - Interactive contact form with notifications

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. Login with any email and password (minimum 3 characters)
4. Explore the dashboard and all pages

## 🎮 Usage Guide

| Page | Description |
|------|-------------|
| **Login** | Enter any email and password (min 3 chars) to access the portal |
| **Home** | View platform overview, features, and key statistics |
| **Dashboard** | View KPI metrics, interactive charts, and analytics |
| **Launch Cognos** | Click button to open IBM Cognos dashboard in new tab |
| **Dark Mode** | Click moon/sun icon in navbar to toggle theme |
| **About** | Learn about the platform and its capabilities |
| **Contact** | Send messages (demo form with notifications) |

## 📱 Responsive Design

- **Desktop (1200px+)** - Full layout with all features visible
- **Tablet (768px - 1199px)** - Adjusted grid layout, readable fonts
- **Mobile (below 768px)** - Collapsible navigation menu, stacked cards

## 📊 Dashboard Charts

The dashboard includes four interactive charts:

1. **Claims by Policy Type** - Bar chart showing distribution across Health, Motor, Life, and Property
2. **Monthly Claims Trend** - Line chart showing claims filed throughout the year
3. **Claim Status Distribution** - Doughnut chart showing Settled, Pending, and Rejected claims
4. **Agent Performance** - Radar chart comparing top 5 agents by performance rating
